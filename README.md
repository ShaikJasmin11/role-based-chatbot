\# Role-Based Chatbot (RISE Internship Project)

\## Description

This project implements a role-based chatbot that changes its responses

based on selected personality roles such as Teacher, Fitness Coach, and HR.

The chatbot uses classical NLP techniques like TF-IDF and Naive Bayes

for intent classification, without relying on external LLM APIs.

\## Technologies Used

\- Python

\- NLTK

\- Scikit-learn

\- Pandas

\## Features

\- Intent classification

\- Role-based response generation

\- Prompt engineering using templates

\- Console-based interaction

\## How to Run

1\. Install dependencies:

pip install -r requirements.txt

2\. Train the model:

python train\_model.py

3\. Run the chatbot:

python chatbot.py
